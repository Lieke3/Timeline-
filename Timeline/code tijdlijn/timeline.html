Responsive Webdesign Timeline

1990 - Het wereldwijde web ontstond

Hierdoor gingen ontwerpers over van papier naar pixels. Het internet is ontwikkeld voor wetenschappers, zodat ze gemakkelijker informatie konden uitwisselen. Voorheen ging dit via systemen en software en kostte dit veel tijd. Tim Berners-Lee ontwikkelde het internet bij CERN.

1996 - Opvallende HTML-technieken

David Siegel publiceert zijn boek Creating Killer Websites. Hierin deelt hij technieken om opvallende ontwerpen in HTML te maken. Een voorbeeld is het gebruiken van een transparante GIF om de whitespace in controle te houden. Daarnaast gaf hij ook het table element als oplossing om data te ordenen.

1999 - Normalisering HTML

HTML4 werd genormaliseerd, waardoor uppercase, lowercase, tag namen en attributen gebruikt kunnen worden. Daarnaast moesten wel alle attributen beschreven worden. Ook hadden alleenstaande elementen, zoals afbeeldingen, een sluitingstag nodig.

2000 - A Dao of Web Design

Online tijdschrift A List Apart publiceert een artikel genaamd A Dao of Web Design. Hier worden nieuwe media die vooruitlopen op een vorig medium getoond. ELk nieuw medium baseert zijn systeem op het vorige medium. Daardoor leken de eerste lay-outs ook op papierontwerpen.

2005 - Opkomst JavaScript

JavaScript kwam op toen Jesse James Garrett het artikel ‘Ajax: A New Approach to Web Applications’ publiceerde. Met JavaScript was het mogelijk voor een browser om data te versturen en ontvangen. Hierdoor kon de gebruiker beter werken en werd de user experience verbeterd.

2007 - iPhone zorgt voor meer responsive design

Steve Jobs brengt de iPhone uit, hiermee kon voor het eerst het wereldwijde web op een mobiel apparaat worden bezocht. Hierdoor werd het gebruiksvriendelijk om websites responsive te maken. Door de jaren heen konden steeds meer sites zich aan verschillende apparaten aanpassen.

2010 - Ontwerpen naar menselijke behoeften

Ethan Marcotte gaf een lezing over architectuur, dit was op An Event Apart in Seattle. De lezing ging over hoe websites ontworpen kunnen worden naar de wensen van de mens. Een wens was dat sites zich aan verschillende apparaten konden aanpassen.

2015 - Afhankelijk van JavaScript

Webdesigners worden meer afhankelijk van JavaScript. NASA lanceerde zijn website in een web app, maar daarvoor hadden gebruikers eerst JavaScript nodig. Voordat websites dan informatie konden opslaan en versturen, moest de gebruiker JavaScript gedownload hebben.
